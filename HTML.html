ALL ABOUT HTML

HTML stands for HyperText Markup Language. It's the foundation for creating web pages and is considered the most fundamental building block of the web. Let's break down what HTML does:

Structure and Content:

HTML uses a set of tags to define the structure and content of a web page. These tags are essentially instructions for web browsers on how to display the information. For instance, you can use tags to create headings, paragraphs, lists, images, and hyperlinks.

Not Visual Design:

It's important to note that HTML itself doesn't directly control the visual design of a webpage. That's the realm of Cascading Style Sheets (CSS), which works alongside HTML to style the elements defined in the HTML code. SOME IMPORTANT POINTS ON HTML TAGS

  1. HTML tags are keywords surrounded by angle brackets like
  2. HTML tags normally come in pairs like and
  3. The start tag is the first in a pair, while the end tag is the second.
  4. Start and end tags are sometimes known as opening and closing tags.
